## Step 1: Set Up Your Wallet

### Option A: MetaMask (Recommended)
1. **Download MetaMask:**
   - Visit [metamask.io](https://metamask.io)
   - Install the browser extension or mobile app
   - Create a new wallet or import existing

2. **Secure Your Wallet:**
   - Write down your 12-word seed phrase
   - Store it offline in a secure location
   - Never share your seed phrase or private keys

3. **Add Base Network:**
   ```
   Network Name: Base
   RPC URL: https://mainnet.base.org
   Chain ID: 8453
   Currency Symbol: ETH
   Block Explorer: https://basescan.org
   ```

### Option B: Coinbase Wallet
1. **Download the App:**
   - App Store or Google Play Store
   - Create account or import existing wallet

2. **Enable Base Network:**
   - Go to Settings ‚Üí Networks
   - Enable Base network
   - You're ready to use $MOTO

### Option C: Trust Wallet
1. **Download Trust Wallet:**
   - App Store or Google Play Store
   - Set up wallet with seed phrase

2. **Add Base Network:**
   - Go to Settings ‚Üí Networks
   - Add custom network with Base details
   - Enable the network

## Step 2: Get ETH for Transactions

### Methods to Get ETH

#### Option 1: Centralized Exchanges
1. **Create Account:**
   - Coinbase, Binance, or Kraken
   - Complete KYC verification

2. **Buy ETH:**
   - Purchase ETH with fiat currency
   - Use debit card, bank transfer, or other payment methods

3. **Withdraw to Wallet:**
   - Send ETH to your wallet address on Base network
   - Use the withdrawal address from your Web3 wallet

#### Option 2: Decentralized Exchanges
1. **Use Uniswap or SushiSwap on Ethereum:**
   - Connect wallet to Ethereum mainnet
   - Swap other tokens for ETH
   - Bridge ETH to Base using the Base Bridge

#### Option 3: Base Bridge
1. **Visit Base Bridge:**
   - Go to [bridge.base.org](https://bridge.base.org)
   - Connect your wallet
   - Bridge ETH from Ethereum to Base
   - Pay the bridging fee

- **Optimal:** 0.1 ETH (for active trading/claiming)
